# riben.life



### Use cases

顧客出發前，先在Google map選店家，訂位取號、瀏覽菜單。依照顯示的排隊狀況，再出發。

到場後，點完餐、付好款， 入座用餐。

顧客也可在出發前，就送出訂單、付款。依號到場時，直接取貨(外帶)，或入座。

店家：收到訂位/訂單時，可與顧客溝通或直接接受訂單。
