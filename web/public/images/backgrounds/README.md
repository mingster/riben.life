# Background images (Avaturn-style)

Place background images here for the virtual experience background switcher. The UI expects these filenames (same design as [Avaturn editor](https://hub.avaturn.me/editor)):

| File        | Preset label |
| ----------- | ------------ |
| `studio.jpg`  | Studio  |
| `outdoor.jpg` | Outdoor |
| `indoor.jpg`  | Indoor |
| `light.jpg`   | Light  |
| `dark.jpg`    | Dark   |

Use JPG or PNG; name must match exactly. If a file is missing, the switcher shows a fallback color and the scene uses a neutral dark background.
